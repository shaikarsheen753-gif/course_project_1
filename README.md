# Bright Future Academy Educational Website

This is a beginner-friendly frontend educational website created for the MANTRA 2026 Summer School Assignment.

## Website Type
Educational Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Courses
- Programs
- Gallery
- Admission
- Contact

## Features
- Responsive design for mobile, tablet and desktop
- SEO tags on every page
- Meta title, description and keywords
- Image alt text for accessibility
- Mobile navigation menu
- Contact form validation using JavaScript
- Admission form validation using JavaScript
- Modern card layout design
- Responsive image gallery
- Internal navigation links
- Sticky navigation bar

## Educational Website Features
- Course information section
- Academic programs section
- Student admission form
- Faculty and campus information
- Student gallery section
- Contact and enquiry form

## How to Run
1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html` in any browser or use Live Server.

## GitHub Hosting Steps
1. Create a GitHub repository.
2. Upload all project files.
3. Commit and push the code to GitHub.

## Netlify Deployment Steps
1. Login to Netlify.
2. Click **Add new site** → **Import an existing project**.
3. Connect your GitHub repository.
4. Select the Bright Future Academy repository.
5. Click **Deploy Site**.
